# BotMentor — A 2-Minute Orientation (for new volunteers & stakeholders)

A quick plain-language briefing you can read aloud at a kickoff or hand to a new team member.

**BotMentor is…** a free website that coaches student robotics teams through troubleshooting,
in the style of a Nebraska Robotics Expo mentor.

**The student does…** types what's wrong with their robot and reads back a ranked list of
likely causes with safe steps to try.

**The tool does NOT…** fix the robot, or claim to have tested anything. It coaches; the
student tests.

**The coaching method (4D) is…**
- *Delegation* — student does the hands-on test.
- *Description* — student learns to describe the problem well.
- *Discernment* — suggestions show confidence and flag the safest check first.
- *Diligence* — safety reminders; honesty when unsure.

**Costs…** free to host; ~$5–10/month if using real AI answers, $0 in practice mode.

**Who owns it after launch…** the organization does. This folder has a plain-language guide
for non-technical owners and a separate technical runbook set for volunteers who can code.
Either way, the site keeps running and can grow.

**The one thing to remember…** our job isn't done when the tool works — it's done when
*someone else can own it.* That's why these two guides exist.
